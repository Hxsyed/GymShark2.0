# GymShark2.0
----------------------------------------------------------------------------------------------------------------------------------------------------------------------
## App Functionality

    ◙ User Register/ Login
  
    ◙ Random generated motivational quotes
  
    ◙ Full Week worth of body workouts
  
    ◙ Built-in countdown timer
    
    ◙ Guest workout option for first time users
